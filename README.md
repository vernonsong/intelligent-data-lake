## 开发流程

1. **需求阶段** - PM 细化需求 → PRD 文档
2. **开发阶段** - Dev 实现功能 → 提交代码
3. **审查阶段** - Reviewer 代码审查 → 反馈修改
4. **测试阶段** - QA 测试验证 → 测试报告
5. **验收阶段** - PM 验收确认 → 发布
